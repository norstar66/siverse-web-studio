# SIVERSE Web Studio

Separate sales-focused site for SIVERSE website services.

## Purpose
A clean, conversion-oriented site for selling:
- starter websites
- standard multi-page sites
- website refreshes
- recurring care plans

This project stays distinct from the main SIVERSE Labs site so product/mission messaging does not muddy the sales story.

## Run locally
```bash
npm install
npm run dev
```
